
## 📦 Projeto 1: Controle de Estoque (Product Stock Manager)

### 📝 Descrição

Aplicação em Java para cadastrar um produto, atualizar a quantidade em estoque e exibir as informações formatadas.

### ⚙️ Tecnologias

* Java 21
* Scanner para entrada de dados
* Programação Orientada a Objetos (POO)

### 🚀 Funcionalidades

✔️ Cadastro do produto (nome, quantidade e preço)
✔️ Atualização de estoque (entrada e saída de produtos)
✔️ Exibição formatada dos dados do produto
